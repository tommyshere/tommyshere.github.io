Pulse

Ruby on Rails

Pulse was done as a 3 member team in roughly 3 days. This web app helps the user in emergency situations. The emergency contacts that are added to the database are notified by text (Twilio API) the user is in trouble and where the user is at (Google Maps API).

Technologies Used: HTML5/CSS/Bootstrap, Ruby, Rails, Google Maps API, Twilio API

Outdoorsy

Python on MVC Framework

Outdoorsy is a web application that allows users to search for and share information about outdoor areas in the world It helps users conenct by notifying each other of advice, hidden gems, and reviews of certain places people want to visit.

Technologies Used: HTML5/CSS3/Bootstrap/jQuery, Python, Pylot (MVC framework developed at Coding Dojo on Flask), MySQL, Google Maps API, Open Weather Map API, Facebook API
